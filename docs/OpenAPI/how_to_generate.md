# どうやって OpenAPI 用の HTML を作成するのか

## 要件

node.js インストールまたは Docker ， WSL 等使って， node を利用出来るようにしておく．

---

## インストール

インストールコマンド

`npm -g install redoc-cli`

## 実行コマンド

`redoc-cli bundle ディレクトリ+ファイル名`
